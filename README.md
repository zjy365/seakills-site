# Seakills Site

Landing site source for Seakills.

## Development

```bash
pnpm install
pnpm dev
```

## Production

```bash
pnpm build
pnpm start
```

## Docker

```bash
docker build -t seakills-site .
docker run -p 3000:3000 seakills-site
```
